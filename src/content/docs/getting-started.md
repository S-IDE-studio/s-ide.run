# はじめに

S-IDEはAIエージェントワークフローに最適化されたインテリジェントな開発環境です。

## インストール

### Windows (推奨)

1. [Releases](https://github.com/S-IDE-studio/S-IDE/releases) から最新のインストーラーをダウンロード
2. `S-IDE-Setup-x.x.x.exe` を実行
3. デスクトップまたはスタートメニューから起動

**注意**: Node.jsがインストールされている必要があります。デスクトップアプリは起動時にバックエンドサーバーを自動的に開始します。

### ソースからビルド

```bash
git clone https://github.com/S-IDE-studio/S-IDE.git
cd S-IDE
npm install
npm run dev:desktop
```

## 基本的な使い方

### ワークスペースの追加

1. サイドバーの「Workspace」ボタンをクリック
2. 「+」ボタンで新しいワークスペースを追加
3. ファイルシステムパスを入力（デフォルトはホームディレクトリ）
4. 「Add Workspace」をクリック

### デッキの作成

1. ターミナルデッキタブの「+」ボタンをクリック
2. デッキ名を入力
3. 関連付けるワークスペースを選択
4. 「Create」をクリック

### ターミナルの使用

各デッキには複数のターミナルを追加できます：

- **ターミナル追加**: デッキヘッダーの「+」ボタン
- **Claudeターミナル**: 「C」ボタンで事前設定されたClaude Codeターミナル
- **削除**: タブの「×」ボタン
- **並び替え**: タブをドラッグ

### Git操作

ソースコントロールパネルで以下の操作が可能：

- ステータス表示
- ファイルのステージ/アンステージ
- コミット
- プッシュ/プル
- ブランチ管理
- Diff表示
